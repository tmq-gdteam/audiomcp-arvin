# audiomcp.ai — Static Deployment

This branch/repo contains the **production-ready static build** of [audiomcp.ai](https://audiomcp.ai).

Deployed via GitHub Pages at: https://tmq-gdteam.github.io/audiomcp-arvin/

## Stack
- React + Vite + Tailwind CSS v4
- Statically built with `vite build`

## Deployment
Automatically deployed via GitHub Actions on every push to `main`.
